WELCOME! This package is the SNAPDRAGON edition, tuned for Snapdragon X NPUs
using the QNN execution provider. Runs entirely on-device. No cloud. No API
costs. Works in airplane mode.

If you are on an Intel Core Ultra device, download
`ZavaInsuranceAI-Intel-vX.Y.Z.zip` instead — it ships an OpenVINO-tuned chain.

FIRST TIME SETUP
════════════════════════════════════════════════════════════════════════════════

  1. Double-click: Setup.bat  (or run Setup.ps1 in PowerShell)
     • Installs Foundry Local if missing
     • Creates a Python virtual environment
     • Installs Flask, OpenAI SDK, and foundry-local-sdk
  2. Double-click: StartApp.bat
     • Sets ZAVA_PLATFORM=snapdragon
     • Launches Flask on http://localhost:5000
     • Opens your browser automatically

SNAPDRAGON-SPECIFIC NOTES
════════════════════════════════════════════════════════════════════════════════

  • Default model chain prefers QNN INT4 NPU builds:
        qwen2.5-1.5b → phi-3-mini-4k → phi-3.5-mini → phi-4-mini
  • ARM64-native Python is strongly recommended (x64 Python runs under
    emulation and silicon detection can mis-report).
  • Foundry Local automatically picks the right QNN variant for your NPU.
  • Check the NPU Dashboard — `hardware` should read "NPU (QNN)".
