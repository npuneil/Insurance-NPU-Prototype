FIRST TIME SETUP
════════════════════════════════════════════════════════════════════════════════

  1. Double-click: Setup.bat  (or run Setup.ps1 in PowerShell)
     • Installs Foundry Local if missing
     • Creates a Python virtual environment
     • Installs Flask, OpenAI SDK, and foundry-local-sdk
  2. Double-click: StartApp.bat
     • Sets ZAVA_PLATFORM=intel
     • Launches Flask on http://localhost:5000
     • Opens your browser automatically

════════════════════════════════════════════════════════════════════════════════
INTEL-SPECIFIC NOTES
════════════════════════════════════════════════════════════════════════════════

  • Default model chain prefers OpenVINO INT4 NPU builds:
        phi-3.5-mini → phi-4-mini → qwen2.5-1.5b → phi-3-mini-4k
  • Foundry Local automatically picks the right OpenVINO variant for your NPU.
  • The first model load may take 60–120 seconds while OpenVINO compiles the
    model graph for the NPU. Subsequent loads are cached and much faster.
  • Check the NPU Dashboard — `hardware` should read "NPU (OpenVINO)".
